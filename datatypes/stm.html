STM

Overview

STM is a monad which describes transactions involving TVars. It is executed via STM.atomically:

import cats.effect.IO
// import cats.effect.IO

import io.github.timwspence.cats.stm.{STM, TVar}
// import io.github.timwspence.cats.stm.{STM, TVar}

val prog: IO[(Int, Int)] = for {
  to   <- TVar.of(0).commit[IO]
  from <- TVar.of(100).commit[IO]
  txn  <- STM.atomically[IO] {
    for {
      balance <- from.get
      _       <- from.modify(_ - balance)
      _       <- to.modify(_ + balance)
    } yield ()
  }
  t   <- to.get.commit[IO]
  f   <- from.get.commit[IO]
} yield f -> t
// prog: cats.effect.IO[(Int, Int)] = IO$898682541

val result = prog.unsafeRunSync
// result: (Int, Int) = (0,100)

Retries

STM.atomically supports the concept of retries, which can be introduced via STM.check:

import cats.effect.IO
// import cats.effect.IO

import io.github.timwspence.cats.stm.{STM, TVar}
// import io.github.timwspence.cats.stm.{STM, TVar}

val to   = TVar.of(1).commit[IO].unsafeRunSync
// to: io.github.timwspence.cats.stm.TVar[Int] = io.github.timwspence.cats.stm.TVar@1ad1086f

val from = TVar.of(0).commit[IO].unsafeRunSync
// from: io.github.timwspence.cats.stm.TVar[Int] = io.github.timwspence.cats.stm.TVar@969b4eb

val txn: IO[Unit]  = STM.atomically[IO] {
  for {
    balance <- from.get
    _       <- STM.check(balance > 100)
    _       <- from.modify(_ - 100)
    _       <- to.modify(_ + 100)
  } yield ()
}
// txn: cats.effect.IO[Unit] = IO$861876472

txn.unsafeRunSync will block until the transaction succeeds (or throws an exception!). Internally, this is implemented by keeping track of which TVars are involved in a transaction and retrying any pending transactions every time a TVar is committed.

OrElse

STM.orElse is built on top of the retry logic and allows you to attempt an alternative action if the first retries:

import cats.effect.IO
// import cats.effect.IO

import io.github.timwspence.cats.stm.{STM, TVar}
// import io.github.timwspence.cats.stm.{STM, TVar}

val to   = TVar.of(1).commit[IO].unsafeRunSync
// to: io.github.timwspence.cats.stm.TVar[Int] = io.github.timwspence.cats.stm.TVar@55fbf2f

val from = TVar.of(0).commit[IO].unsafeRunSync
// from: io.github.timwspence.cats.stm.TVar[Int] = io.github.timwspence.cats.stm.TVar@7b2eb567

val transferHundred: STM[Unit] = for {
  b <- from.get
  _ <- STM.check(b > 100)
  _ <- from.modify(_ - 100)
  _ <- to.modify(_ + 100)
} yield ()
// transferHundred: io.github.timwspence.cats.stm.STM[Unit] = io.github.timwspence.cats.stm.STM@3b743587

val transferRemaining: STM[Unit] = for {
  balance <- from.get
  _       <- from.modify(_ - balance)
  _       <- to.modify(_ + balance)
} yield ()
// transferRemaining: io.github.timwspence.cats.stm.STM[Unit] = io.github.timwspence.cats.stm.STM@32489a08

val txn  = for {
  _    <- transferHundred.orElse(transferRemaining)
  f    <- from.get
  t    <- to.get
} yield f -> t
// txn: io.github.timwspence.cats.stm.STM[(Int, Int)] = io.github.timwspence.cats.stm.STM@2fa92a82

val result = txn.commit[IO].unsafeRunSync
// result: (Int, Int) = (0,1)

Aborting

Transactions can be aborted via STM.abort:

import cats.effect.IO
// import cats.effect.IO

import cats.syntax.flatMap._
// import cats.syntax.flatMap._

import io.github.timwspence.cats.stm.{STM, TVar}
// import io.github.timwspence.cats.stm.{STM, TVar}

val to   = TVar.of(1).commit[IO].unsafeRunSync
// to: io.github.timwspence.cats.stm.TVar[Int] = io.github.timwspence.cats.stm.TVar@789e9b8b

val from = TVar.of(0).commit[IO].unsafeRunSync
// from: io.github.timwspence.cats.stm.TVar[Int] = io.github.timwspence.cats.stm.TVar@24718a2b

val txn  = for {
  balance <- from.get
  _       <- if (balance < 100)
               STM.abort(new RuntimeException("Balance must be at least 100"))
             else
               STM.unit
  _ <- from.modify(_ - 100)
  _ <- to.modify(_ + 100)
} yield ()
// txn: io.github.timwspence.cats.stm.STM[Unit] = io.github.timwspence.cats.stm.STM@29cacac7

val result = txn.commit[IO].attempt.unsafeRunSync
// result: Either[Throwable,Unit] = Left(java.lang.RuntimeException: Balance must be at least 100)

Note that aborting a transaction will not modify any of the TVars involved.
