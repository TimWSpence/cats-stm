TVar

A TVar (transactional variable) is a mutable memory location that can be be read and modified via STM actions.

import cats.effect.IO
// import cats.effect.IO

import io.github.timwspence.cats.stm._
// import io.github.timwspence.cats.stm._

val to   = TVar.of(0).commit[IO].unsafeRunSync
// to: io.github.timwspence.cats.stm.TVar[Int] = io.github.timwspence.cats.stm.TVar@5ce25eac

val from = TVar.of(100).commit[IO].unsafeRunSync
// from: io.github.timwspence.cats.stm.TVar[Int] = io.github.timwspence.cats.stm.TVar@60bb7996

val txn: STM[(Int, Int)] = for {
  balance <- from.get
  _       <- from.modify(_ - balance)
  _       <- to.modify(_ + balance)
  res1    <- from.get
  res2    <- to.get
} yield res1 -> res2
// txn: io.github.timwspence.cats.stm.STM[(Int, Int)] = io.github.timwspence.cats.stm.STM@4b53f307

val result = txn.commit[IO].unsafeRunSync
// result: (Int, Int) = (0,100)

Note that this does not modify either from or to!! It merely describes a transaction which must be executed via STM.atomically
